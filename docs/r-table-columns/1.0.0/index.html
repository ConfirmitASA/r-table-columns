r-table-columns

TableColumns builds an array of columns that contains column information in the ro indexed by defaultHeaderRow which is passed in method configuration. A defaultHeaderRow is to be considered as a target row for actions to take place, like sorting or filtering: clicking on those column headers in the row would perform the action.
